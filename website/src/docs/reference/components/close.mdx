# Close

The button that closes the drawer.

## API Reference

| Prop      | Type      | Default |
| --------- | --------- | ------- |
| `asChild` | `boolean` | `false` |

## Radix UI Integration

This component extends [Radix UI Dialog.Close](https://www.radix-ui.com/primitives/docs/components/dialog#close) and inherits all its props.

## Props

### asChild

Change the default rendered element for the one passed as a child, merging their props and behavior.

```tsx
<Drawer.Close asChild>
  <button className='custom-close-button'>
    <CloseIcon />
  </button>
</Drawer.Close>
```

## Usage

The close button can be placed anywhere within the drawer content:

```tsx
<Drawer.Content>
  <Drawer.Close className='absolute top-4 right-4'>Close</Drawer.Close>
  <Drawer.Title>Drawer Title</Drawer.Title>
  <div>{/* Drawer content */}</div>
  <Drawer.Close className='mt-4'>Cancel</Drawer.Close>
</Drawer.Content>
```

You can also close the drawer programmatically by managing the `open` state on the `Drawer.Root`:

```tsx
const [open, setOpen] = useState(false)

<Drawer.Root open={open} onOpenChange={setOpen}>
  <Drawer.Content>
    <button onClick={() => setOpen(false)}>
      Close Programmatically
    </button>
  </Drawer.Content>
</Drawer.Root>
```
